pixlr.com


Pixlr is known to me since 2012 and the purpose of the post is informed, voluntary recommendation üôÇ. It serves me to remove the background, edit a photo or create new one from collected clippings. As for the advantages, first of all it does not require any software installation. It allows to load images directly from URL. Project can be saved in special format to store it's structure and reopen it. Pixlr contains nice effects, stickers, brush for overwriting specified color, cutting out (including lasso). We have versions X (Express) and E (Editor) depending on whether we need to make quick modification (X) or work on advanced project (E). Free version can be optionally extended with premium package which isn't invasive. Some elements marked as premium have been prepared in such way that in the free version you have to wait a while to be able to use them in project.
